title: How to Learn the Platform
---

This guide gives a practical order for learning the PCA platform.

## Suggested learning path

1. Start with the [Platform & Services](../index.md) overview.
2. Learn [PCA Explore](../pca-explore/index.md) so you can see what already exists.
3. Continue to [PCA Consume](../pca-consume/index.md) if you need to retrieve or integrate content.
4. Move to [PCA Evolve](../pca-evolve/index.md) if you need to create or extend content.
5. Read [PCA Govern](../pca-govern/index.md) if you participate in review workflows.
6. Read [PCA Host](../pca-host/index.md) if you need PCA to host content you already have.

## General advice

- Learn the content ecosystem before creating anything new.
- Use the human-readable platform views before building API integrations.
- Treat review and hosting as part of the lifecycle, not as afterthoughts.
