---
title: Digital Symbol Editor Submission Workflow
---

This guide explains the actual platform workflow after you have prepared the SVG.

## Step-by-step

1. Open the Symbol Editor in the platform.
2. Upload the prepared geometry-only SVG file.
3. Add the metadata that explains what the symbol represents.
4. Add connection points if the symbol requires them.
5. Define the center of rotation where relevant.
6. Review the symbol data before submission.
7. Save and submit the symbol for review.

## What to prepare before you upload

Prepare these before you start the form:

- the final SVG file
- the intended symbol name
- the description or definition
- any additional context that reviewers need to understand the proposal
- the connection points and rotation center if the symbol should support them

## What happens after submission

After submission, the symbol becomes part of the normal review lifecycle. Reviewers can inspect it, assess impact, and decide whether it should move forward.
