`PCA Evolve` helps users create, propose, and extend content through controlled workflows.

Use this area when you need to:

- create a new ontology or reference-data context
- extend existing content such as CFIHOS
- create IMF types
- propose digital symbols
- decide which evolve workflow best matches your need

## Before you create anything

1. Use [PCA Explore](../pca-explore/index.md) to check whether the content already exists.
2. Confirm whether you need to create a new item, a new version, or a structured extension.
3. Make sure you have the required labels, descriptions, justification, and target collection or ontology context.
